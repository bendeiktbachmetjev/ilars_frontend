# Google Play Store — ready-to-use assets

This folder contains everything you can copy or host when publishing ABBA LARS to the Play Store.

| File | Use |
|------|-----|
| **store_listing_copy.txt** | Copy app name, short description (80 char), and full description into Play Console → Store listing. |
| **privacy_policy.html** | Host this file on your website or GitHub Pages; put the public URL in Play Console → App content → Privacy policy. |
| **app_content_answers.md** | Suggested answers for Data safety, Health declaration, Ads, Content rating, Target audience. |
| **build_release.sh** | From app root: `./play_store/build_release.sh` to build the release AAB. |

See **PLAY_STORE_PUBLISHING.md** in the app root for the full step-by-step guide.
